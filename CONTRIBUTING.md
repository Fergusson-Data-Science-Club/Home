## How to contribute
1. Fork the repository.
   
2. Clone:
git clone https://github.com/<your-username>/<repo-name>.git
cd <repo-name>

3. Create a feature branch:
git checkout -b feat/your-feature

4. Make changes and commit:
git add .
git commit -m "Add: short description"
git push origin feat/your-feature

5. Open a Pull Request to `main` with a clear description and any screenshots or replication steps.
## Code style
- Use clear variable names and comments.
- Add README updates for major features.
- If adding notebooks, include a short `README.md` inside the notebook folder explaining dataset and steps.

## Communication
- For questions, ping the maintainers in the club group or open an issue.
